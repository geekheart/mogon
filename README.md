# 墨攻

这里是 `mogon` 仓库的总览页，用来集中收录我自己写的各种小脚本。

## 脚本总览

### GitHub SSH 一键配置

作用简介：

- 在 Linux 或 WSL 下生成或复用 GitHub SSH 密钥
- 启动 `ssh-agent`
- 自动把公钥复制到剪贴板
- 打开 GitHub 新增 SSH Key 页面

一键调用：

```bash
curl -fsSL https://raw.githubusercontent.com/geekheart/mogon/main/github/github_ssh_setup.sh | bash
```

[查看详情](./github/README.md)

## 使用方式

1. 在上面的总览里找到你要的脚本
2. 直接复制“一键调用”命令执行
3. 如果想看参数、步骤和注意事项，点击“查看详情”
